Submitting a Job to the DPC

Using XMP File Info Panels (DPC Job Ticket)

Our custom XMP File Info Panel, DPC Job Ticket, can be found on all CIAS computers within all Adobe Creative Suite applications and Adobe Acrobat Professional. This custom XMP Info Panel allows you to attach your job ticket information directly to your file. When uploading this file to SPM-Portal DPC Dropbox volume, the job ticket information fields will automatically synchronize to the File Info Panel information.

  1. Before the file is exported to a PDF, go to File > File Info.

  2. On the left side of the window, select the DPC Job Ticketpanel.

  3. Complete all fields in the job ticket. If you have a question about any of the fields, visit us in DPC (A-161) or call us at 475-5954.

  4. Once you have entered all of your information into the ticket, click OK.

  5. Save your file. Avoid special characters and spaces in your file name.

  6. To create the PDF, go to File > Export. For optimum output quality, use the High Quality Print PDF Preset setting.

  7. Using Firefox, login with your DCE user on the SPM-Portal website ( spm-portal. rit.edu ). Upload the files following the normal submission directions. The only difference will be that the information fields in the job ticket will already be entered. At that point, these fields should be reviewed, Apply Changes should be selected if changes have been made to the job ticket information, and the Submit Job button should be selected to actually submit the job.
