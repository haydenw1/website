Creating One PDF from Mutliple PDFs

Important Facts

For DPC job submission, rather than having to fill out several job tickets, for PDFs that have the same printing specifications (i.e. sheet size, color/B&W, simplex/duplex, orientation, scaling), it can be more efficient to combine all PDFs into one PDF. This will also speed up processing time in the DPC.

This user guide is intended to instruct you on how to combine multiple PDFs into one PDF file. Follow these simple instructions using Adobe Acrobat 8.

How to Create One PDF from Multiple PDFs Using Acrobat 8

  1. Go to: File > <Create PDF></Create> From Multiple Files...

    Figure 1: Locating the option to combine multiple PDFs in Acrobat

  2. Select Add Files..., add the PDFs to be combined, select the desired quality (at bottom), and click Next.

    Figure 2: Adding the files to be combined

  3. Choose the option (on top) to merge the PDFs into one file (instead of creating a PDF Package) and click Create.

    Figure 3: Selecting the option to merge (or combine) the PDFs

  4. After files have been merged, click Save. Remember to name your file with a unique name (i.e. Stoll_Resumes.pdf).

    Figure 4: Saving the final PDF

  5. Open the file in Acrobat to confirm that the files were combined correctly in the desired sequence and orientation.
