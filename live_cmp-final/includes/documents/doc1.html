Creating One PDF from Multiple PDFs

Rather than having to fill out several job tickets, for PDFs that have the same printing specifications (i.e. sheet size, color/B&W, simplex/duplex, orientation, scaling), it is more efficient to combine all PDFs into one. This will also speed up processing time in the DPC. Follow these simple instructions for Adobe Acrobat.

  1. Go to: File>Create PDF>From Multiple Files...

  2. Select Add Files...

  3. Select the PDFs that should be combined.

  4. Create the PDF.

    In Acrobat 8, be sure to use the option to merge files into a single PDF as opposed to assembling files into a PDF Package.

  5. Upload to SPM-Portal at spm-portal.rit.edu.
