Loading DPC Job Ticket Panel to Personal Computers

Our custom XMP Info Panel has been uploaded to all CIAS computers. However, if you would like to use the panel on your personal computer, follow these instructions:

  1. Login into SPM-Portal ( spm-portal.rit.edu) with your DCE user.

  2. Select the Software Plug-Ins volume.

  3. Select the DPC_File_Info_Panels folder.

  4. If you are running Adobe Creative Suite (2 or 3) on Mac OSX, select Download next to “DPC_JobTicket_OSX_vXX.txt”. If you are running Creative Suite (2 or 3) on Windows XP or Vista, select Download next to “DPC_JobTicket_XP_vXX.txt”.

  5. Drop the text file into the Custom File Info Panels folder on your hard drive:

    Mac OSX: Hard Disk/Library/Application Support/Adobe/XMP/ Custom File Info Panels/

    Windows Vista: C:\Users\<your user></your>\AppData\Roaming\Adobe\XMP\ Custom File Info Panels\...

    Windows XP: C:\Documents and Settings\All Users\Application Data\Adobe\ XMP\Custom File Info Panels\...

  6. The DPC Job Ticket custom Info Panel will now be found in every Adobe Creative Suite application and Adobe Acrobat Professional.
