Saving and Applying Metadata Templates

If you are working with our custom DPC Job Ticket Info Panel, a simple way to produce your metadata is to use saved metadata templates. Use the following instructions to save and apply metadata templates. Reference Using XMP File Info Panel (DPC Job Ticket) for instructions on how to find and use our custom panel.

Saving metadata templates:*

  1. In Adobe InDesign, Illustrator, or Photoshop, before the file is exported to a PDF, go to File > File Info. In Adobe Acrobat, with your PDF file open, go to File > Properties. In the Properties window, click the Additional Metadata button.

  2. Fill out all fields in the DPC Job Ticket Info Panel.

  3. Click the arrow in the upper right corner of the window.

  4. Click Save Metadata Template... and save the template with a useful name (i.e. DigitalColor_LetterSize).

  5. Click OK to apply your metadata.

Applying metadata templates:

  1. In Adobe InDesign, Illustrator, or Photoshop, before the file is exported to a PDF, go to File > File Info. In Adobe Acrobat, with your PDF file open, go to File > Properties. In the Properties window, click the Additional Metadata button.

  2. Click the arrow in the upper right corner of the window.

  3. Click the name of the template you would like to apply. To see the root of your templates, select Show Templates... and find your saved template within XMP>Metadata Templates.

  4. The template can then be used as is or modified accordingly.

  5. Click OK to apply your metadata.

*You should save your metadata templates onto your personal computer as RIT computers clear files each night.
